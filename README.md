# Chromium GoTo Line

Chrome Extension that allows the user to go to a line in
https://source.chromium.org/ by typing Control+Comma+<number>+Enter

### How to build

1. Clone repo
2. Go to chrome://extensions
3. Click on 'Load unpacked' and select the extension
